== Description ==

= Zintegruj Woocommerce z Freebee =

Wtyczka pozwoli na powiązanie złożonych na Twojej stronie zamówień z programem Freebee. Podczas składania zamówienia użytkownicy będą mieli możliwość wpisania swoich danych.

== Installation ==

1. Pobierz i rozpakuj folder z plikami wtyczki do folderu z pozostałymi wtyczkami na Twoim serwerze
2. Aktywuj wtyczkę w panelu
3. Przejdź na stronę ustawień wtyczki i uzupełnij 2 pola, następnie naciśnij 
4. Gotowe!

= Minimum Requirements =

* PHP version 5.4 or greater (PHP 5.6 or greater is recommended)
* MySQL version 5.2 or greater (MySQL 5.6 or greater is recommended)
* Woocommerce 2.6

== Frequently Asked Questions ==

= Czego potrzebuję, aby wtyczka działała? =

Potrzebujesz włączonej wtyczki Woocommerce i swpoch danych do nawiązania połączenia z Freebee.

= Widzę informację o poprawnym połączeniu ale zamówienia klientów nie trafiają do Freebee =

Na stronie ustawień wtyczki Freebee są przyciski kierujące do logów - znajdziesz tam informacje dotyczące wyłapanych komunikatów. W kwestii dalszej pomocy napisz do nas przez zakładkę pomocy technicznej na stronie wtyczki.

= Skąd mogę uzyskać swoje dane do włączenia wtyczki? =

Potrzebujesz konta na Freebee.pl, tam też uzykasz swoje dane.
